# ASAIAS Publication Registry

Manifest, counters and publication accounting layer for ASAIAS.

## Primary Role

Track what was drafted, approved, published, indexed, corrected and archived.

## Baseline Contents

- release manifests
- approval and publication status boards
- registry entries
- publication accounting notes

## Operating Rule

**Human mandate. AI execution. Open research. Human approval.**

**Человеческий мандат. AI-исполнение. Открытые исследования. Человеческое утверждение.**

## Repository Rules

- Registry entries must stay synchronized with public releases.
- Use clear status transitions.
- Keep archive references durable and explicit.

## Approval Gate

Nothing public is final without human review and human approval.
