## Release Logic

The first cycle follows one clear sequence:

1. Explain why strategic AI management is an institutional literacy requirement.
2. Show how an academy becomes AI-native without fake scale or tool hype.
3. Define responsibility contours for autonomous systems as a research baseline.
4. Publish a frozen weekly governance index release.
5. Add a separate rating interpretation layer without rewriting the index itself.

This sequence keeps narrative, method, evidence, ranking and interpretation distinct while still belonging to one release family.

2. register corrections as follow-on entries rather than silent rewrites;
